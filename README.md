## 🕷️ What's this?

A Spider-Gwen themed personal portfolio built with React + Vite. Dark, moody, and pink — just like Ghost-Spider herself. Every section was crafted with intention, from the glitchy name effect to the floating spider that watches over the hero page.

---

## ✨ Features

- **🕷️ Animated Spider** — A glowing white spider hangs in the hero section, floating on an invisible web thread. Because why not.
- **⚡ Glitch Text Effect** — The name hits different with a layered glitch animation in pink and blue.
- **⌨️ Typewriter Roles** — Cycles through roles like `Frontend Developer`, `UI/UX Designer`, `Creative Coder` and more.
- **🛠️ Tools & Languages Section** — A full grid of tech icons (HTML, CSS, JS, React, Python, C++, and more) with hover glow effects in each tool's brand color.
- **📊 Animated Skill Bars** — Skill bars animate in when you scroll to them.
- **🌐 Web Pattern Background** — Subtle spider-web SVG pattern tiled across the entire site.
- **📱 Responsive** — Photo frame and spider hide gracefully on smaller screens so nothing overlaps.
- **🔗 Clickable Projects** — Project cards link directly to GitHub repos.
- **📄 CV Viewer** — Opens CV in a new tab so visitors can read before downloading.
- **🧭 Vertical Side Nav** — Smooth scroll navigation with active state glow and hover tooltips.

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/lavcharms/lavcharms.github.io.git

# Go into the folder
cd lavcharms.github.io

# Install dependencies
npm install

# Start dev server
npm run dev
```

Then open `http://localhost:5173` in your browser.

---

## 📦 Deploy

```bash
npm run deploy
```

Builds the project and pushes to the `gh-pages` branch automatically.
